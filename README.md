Ruby bindings for the eBanc API
===============================

Installation
------------

Ebanc Ruby bindings are available as a gem, to install it just install the gem:

    gem install ebanc

If you're using Bundler, add the gem to Gemfile.

    gem 'ebanc'
Run `bundle install`.

Requirements
------------

* Ruby 1.9.3 or above.
* uri, open-uri, net/http, net/https, json
